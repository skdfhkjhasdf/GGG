# Лабораторная работа 1 — Анализ успеваемости

Небольшая учебная работа по обработке массива оценок и вычислению базовой статистики.

## Задание
1. Считать оценки студентов из CSV.
2. Посчитать среднее, медиану и стандартное отклонение.
3. Сформировать список студентов, чья оценка выше среднего.

## Запуск
```bash
python3 src/lab1.py data/grades.csv
```

## Формат входных данных
CSV с заголовком: `name,grade`.

Пример:
```
name,grade
Иван,4
Анна,5
Пётр,3
```

## Результат
Скрипт печатает статистику и список студентов с оценкой выше среднего.
