# Отчёт по лабораторной работе 1

## Цель
Закрепить навыки чтения CSV, вычисления статистики и формирования отчёта по данным.

## Входные данные
Файл `data/grades.csv` со списком студентов и их оценок.

## Ход работы
1. Считаны данные из CSV с помощью `csv.DictReader`.
2. Рассчитаны среднее, медиана и стандартное отклонение (population stdev).
3. Сформирован список студентов, чья оценка выше среднего.

## Результат
Скрипт выводит статистику и список студентов с оценкой выше среднего!

## Вывод
Задача выполнена, освоены базовые методы статистики и обработка CSV.
